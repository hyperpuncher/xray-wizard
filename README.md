## Install

```sh
curl -sOL https://github.com/hyperpuncher/xray-wizard/raw/refs/heads/main/xray-wizard
chmod +x xray-wizard
```

## Usage

```sh
Usage: xray-wizard [options]

Options:
 -a <client_name>  Add new client
 -r <client_name>  Remove client
 -l                List clients
 -i                Setup xray
```
